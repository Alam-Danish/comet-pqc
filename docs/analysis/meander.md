# Meander

Calculates specific resistance per square.

Type: `meander`

## Parameters

| Name | Type | Default | Description |
|------|------|---------|-------------|
|`cut_param`| `float` | `1e-5` | cut on 1st derivative to id voltage regions |

## Limits

| Name | Type | Description |
|------|------|-------------|
|`r` | `float` | specific resistance per square |
|`r_value` | | |
